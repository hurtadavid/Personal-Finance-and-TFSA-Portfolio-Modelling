# Personal Finance and TFSA Portfolio Modeling
## Overview
This project uses Python to model personal finance and investment growth under different assumptions. It was created ti explore how factors such as starting capital, annual contributions, investment returns, and time horizons affect long-term portfolio growth
## Features
- Projects investment portfolio growth over time
- Compares different annual return scenarios
- Displays results using tables and graphs
- Demonstrates the effect of compound growth
## Tools used
- Python
- pandas
- matplotlib
- Google Colab
- Jupyter Notebook
- GitHub
## Skills
- Financial modelling
- Compound Growth Calculator
- Scenario Analysis
- Data organization using pandas
- Data visualization using matplotlib
- Writing reusable Python functions
- Interpreting longterm investment outcomes
